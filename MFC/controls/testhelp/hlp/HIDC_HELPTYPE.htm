Type of Help to Use

 

Help in a Popup Window

 

With this type of help selected, if a user presses F1 while the Testhelp control has the focus, help will be displayed for the control in a popup window.

 

Show the Whole Help File

 

With this type of help selected, if a user presses F1 while the Testhelp control has the focus, the contents page will be displayed in the main Windows help window.
